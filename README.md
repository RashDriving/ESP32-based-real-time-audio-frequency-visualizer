\# Real-Time audio Frequency Visualizer



ESP32-based real-time music frequency visualizer using FFT and an ILI9341 TFT display.



\## Features



\- 16-band FFT spectrum analyser

\- Top 2 dominant frequency detection

\- Bass / Mid / Treble classification

\- Real-time visualization



\## Hardware



\- ESP32

\- ILI9341 TFT

\- Electret Microphone

\- LM358



\## Libraries



\- TFT\_eSPI

\- arduinoFFT



\## Output



\- Dominant Frequency #1

\- Dominant Frequency #2

\- 16 Band Spectrum Analyzer
